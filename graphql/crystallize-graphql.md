# Crystallize GraphQL API

Read-optimized GraphQL API for fetching structured product data, shapes, topics, price variants, stock, and rich content for storefronts. Queries are scoped to a tenant identifier and return path-based catalogue content.

## Crystallize Catalogue API

**Endpoint:** https://api.crystallize.com

**Documentation:** https://crystallize.com/learn/developer-guides/catalogue-api

**References:**

- Documentation: https://crystallize.com/learn/developer-guides/catalogue-api
- Playground: https://crystallize.com/learn/developer-guides/query-explorer/graphql-playground

## Crystallize Discovery / Search API

Semantic GraphQL API combining browse, search, and autocomplete in a single endpoint. Supports filtering, faceting, ranking, and rich product discovery experiences for storefronts.

**Endpoint:** https://api.crystallize.com

**Documentation:** https://crystallize.com/learn/developer-guides/api-overview

**References:**

- Documentation: https://crystallize.com/learn/developer-guides/api-overview

## Crystallize Shop API

Edge-distributed GraphQL API for cart, promotions, and checkout flows. Optimized for low-latency storefront writes during the buying journey.

**Endpoint:** https://shop-api.crystallize.com

**Documentation:** https://crystallize.com/docs/developer/apis/shop-api

**References:**

- Documentation: https://crystallize.com/docs/developer/apis/shop-api

## Crystallize Order API

GraphQL API for creating, reading, and managing orders linked to a Crystallize tenant. Used by storefronts and back-office tooling to persist completed transactions.

**Endpoint:** https://api.crystallize.com

**Documentation:** https://crystallize.com/learn/developer-guides/order-api

**References:**

- Documentation: https://crystallize.com/learn/developer-guides/order-api

## Crystallize Subscription API

GraphQL API for managing recurring orders, subscription contracts, and renewal events for Crystallize-powered commerce experiences.

**Endpoint:** https://api.crystallize.com

**Documentation:** https://crystallize.com/learn/developer-guides/api-overview

**References:**

- Documentation: https://crystallize.com/learn/developer-guides/api-overview

## Crystallize Core / PIM API

Read/write GraphQL API for managing the product information model: shapes, items, topics, language, price variants, stock locations, and tenant configuration. Powers PIM tooling and back-office automation.

**Endpoint:** https://pim.crystallize.com/graphql

**Documentation:** https://crystallize.com/docs/developer/apis

**References:**

- Documentation: https://crystallize.com/docs/developer/apis
